## My manga history
The extension saves your manga reading history in [Union Mang√°s](https://unionleitor.top/). Open the manga page or chapter and click on the extension to automatically save or list. The extension save only the newest chapter from each manga.


### Installing
To add the extension, follow the steps above:

- Download the extension [here](https://github.com/locozila/Mymangas/archive/master.zip)
- Unzip files
- Open chrome extensions (chrome://extensions/)
- Active Developer Mode
- Load unpacked directory
